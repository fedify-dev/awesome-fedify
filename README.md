Awesome Fedify
==============

A curated list of projects, examples, articles, and talks related to [Fedify],
a TypeScript library for building ActivityPub servers.

Entries are limited to projects and resources that are actively maintained and
genuinely useful to people building on the fediverse with Fedify.

[Fedify]: https://fedify.dev/


Applications
------------

 -  [Ghost]: A publishing platform that uses Fedify for its ActivityPub
    implementation.
 -  [Hollo]: Federated single-user microblogging software.
 -  [Hackers' Pub]: An ActivityPub-enabled social network for hackers.
 -  [Encyclia]: A service that brings researcher profiles and publications from
    ORCID to the fediverse.
 -  [Typo Blue]: A Korean text-only blogging platform with opt-in ActivityPub
    integration.
 -  [SiliconBeest]: A serverless fediverse platform built on Cloudflare
    Workers.

[Ghost]: https://ghost.org/
[Hollo]: https://docs.hollo.social/
[Hackers' Pub]: https://hackers.pub/
[Encyclia]: https://encyclia.pub/
[Typo Blue]: https://typo.blue/
[SiliconBeest]: https://github.com/SJang1/siliconbeest
